# Saju Portfolio

A modern, responsive personal portfolio website showcasing skills, projects, and experience.

## Features

- **Hero Section**: Eye-catching introduction with personal branding
- **About**: Personal background and information
- **Skills**: Technical skills and expertise
- **Projects**: Showcase of completed projects
- **Education**: Academic background and qualifications
- **Journey**: Professional experience and career path
- **Contact**: Contact information and social links
- **Responsive Design**: Optimized for all devices

## Technologies Used

- **React**: Frontend framework for building user interfaces
- **Vite**: Fast build tool and development server
- **Tailwind CSS**: Utility-first CSS framework for styling
- **PostCSS**: CSS processing tool
- **Autoprefixer**: CSS vendor prefixing

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Ahsan-1015/saju-portfolio.git
   cd saju-portfolio
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`

## Build for Production

To build the project for production:

```bash
npm run build
```

The built files will be in the `dist` directory.

## Preview Production Build

To preview the production build locally:

```bash
npm run preview
```
